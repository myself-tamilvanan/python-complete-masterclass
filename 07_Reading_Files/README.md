# Chapter 7: Reading Files

## Overview
Programs often need to read data from files. Python makes file I/O straightforward with built-in functions.

## Topics Covered
- Opening and closing files
- Reading entire file vs line by line
- Counting lines and words
- Searching through files
- Skipping blank lines
- Reading specific parts of a file
- Working with file paths

## Key Concepts
- **open()**: Opens a file and returns a file object
- **File Handle**: An object used to read/write files
- **Newline character (\n)**: End of line marker in text files
- **strip()**: Remove whitespace including newlines
- **with statement**: Automatically closes file when done

## File Open Modes
| Mode | Description              |
|------|--------------------------|
| "r"  | Read (default)           |
| "w"  | Write (overwrites)       |
| "a"  | Append (adds to end)     |
| "r+" | Read and write           |
