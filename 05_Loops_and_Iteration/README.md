# Chapter 5: Loops and Iteration

## Overview
Loops allow programs to repeat code multiple times. This is one of the most powerful features of programming — automating repetitive tasks.

## Topics Covered
- while loops (indefinite loops)
- for loops (definite loops)
- Loop control: break and continue
- Loop idioms (counting, summing, finding)
- Nested loops
- The range() function
- Iterating over strings

## Key Concepts
- **while loop**: Repeats while a condition is True
- **for loop**: Iterates over a sequence of items
- **break**: Exits the loop immediately
- **continue**: Skips to the next iteration
- **range()**: Generates a sequence of numbers
- **Infinite loop**: A loop that never ends (use break to escape)
