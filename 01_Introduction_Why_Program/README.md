# Chapter 1: Introduction - Why Program?

## Overview
This chapter introduces why programming is important and how computers work at a high level.

## Topics Covered
- What is programming and why learn it?
- Hardware architecture (CPU, RAM, Storage, Input/Output)
- Python as a programming language
- Setting up Python on Windows and Mac
- Running your first Python program

## Key Concepts
- **Program**: A set of instructions for a computer
- **Hardware**: Physical components (CPU, Memory, Disk, Network, I/O)
- **Python**: An interpreted, high-level programming language
- **Interactive Mode**: Running Python line by line in the shell
- **Script Mode**: Running a saved .py file

## Prerequisites
None! This is the starting point.

## Setup
1. Download Python 3 from https://www.python.org
2. Install Python (check "Add to PATH" on Windows)
3. Open terminal/command prompt
4. Type `python3` to start interactive mode

## Learning Outcomes
By end of this chapter you will:
- Understand why programming is a valuable skill
- Know how computers work at a basic level
- Be able to install and run Python
- Use Python interactive shell
