# Chapter 49: Sets

## Overview
Sets are unordered collections of unique elements. They are ideal for membership testing, removing duplicates, and mathematical set operations.

## Topics Covered
- Creating sets
- Set methods: add, remove, discard, pop, clear
- Set operations: union, intersection, difference, symmetric difference
- Frozen sets
- Set comprehensions
- Performance advantages of sets

## Key Concepts
- **Set**: Unordered, mutable, no duplicates, O(1) membership test
- **frozenset**: Immutable set (can be used as dict key)
- **Union (|)**: All elements from both sets
- **Intersection (&)**: Elements in both sets
- **Difference (-)**: Elements in first but not second
- **Symmetric Difference (^)**: Elements in either but not both
