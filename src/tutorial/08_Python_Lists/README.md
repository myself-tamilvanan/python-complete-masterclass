# Chapter 8: Python Lists

## Overview
Lists are ordered, mutable collections of items. They are one of the most versatile and commonly used data structures in Python.

## Topics Covered
- Creating and accessing lists
- List indexing and slicing
- List methods (append, pop, sort, etc.)
- Mutable vs. immutable
- List operations
- Lists and strings
- Guardian pattern with lists
- List comprehensions (bonus)

## Key Concepts
- **List**: An ordered, changeable collection of items
- **Index**: Position of an item (starts at 0)
- **Mutable**: Lists can be changed in place
- **append()**: Add item to end of list
- **in operator**: Check if item is in list
- **sorted()**: Returns sorted copy; sort() sorts in place

## Common List Methods
| Method         | Description                     |
|---------------|---------------------------------|
| append(x)     | Add x to end                    |
| insert(i, x)  | Insert x at index i             |
| pop()         | Remove and return last item     |
| pop(i)        | Remove and return item at i     |
| remove(x)     | Remove first occurrence of x    |
| sort()        | Sort in place                   |
| reverse()     | Reverse in place                |
| count(x)      | Count occurrences of x          |
| index(x)      | Find index of first x           |
