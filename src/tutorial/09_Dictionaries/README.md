# Chapter 9: Dictionaries

## Overview
Dictionaries store data as key-value pairs. They are perfect for representing mappings, counting occurrences, and organizing structured data.

## Topics Covered
- Creating dictionaries
- Accessing and modifying values
- Dictionary methods (keys, values, items)
- Checking for keys with in and get()
- Counting patterns with dictionaries
- Counting word frequency
- Most common items

## Key Concepts
- **Dictionary**: An unordered collection of key-value pairs
- **Key**: A unique identifier (usually a string or number)
- **Value**: The data associated with a key
- **get()**: Safely retrieve value with a default
- **items()**: Returns all key-value pairs
- **keys()**: Returns all keys
- **values()**: Returns all values

## Dictionary vs. List
| Feature     | List         | Dictionary          |
|-------------|--------------|---------------------|
| Index       | Integer      | Any hashable type   |
| Ordered     | Yes          | Yes (Python 3.7+)   |
| Mutable     | Yes          | Yes                 |
| Duplicates  | Allowed      | Keys must be unique |
