# Chapter 51: Itertools

## Overview
The itertools module provides fast, memory-efficient tools for creating iterators for looping. These are building blocks for complex iteration patterns.

## Topics Covered
- product: Cartesian product
- permutations: All orderings
- combinations: All unique selections
- combinations_with_replacement
- accumulate: Running totals
- groupby: Group consecutive elements
- chain: Chain iterables together
- islice: Slice an iterator

## Key Concepts
- **Lazy evaluation**: itertools produces results on demand (memory efficient)
- **Infinite iterators**: count(), cycle(), repeat()
- **Combinatoric iterators**: product, permutations, combinations
- **Groupby**: Requires sorted input to work correctly
