# Chapter 6: Strings

## Overview
Strings are sequences of characters. Python provides powerful tools for working with text data — one of the most common tasks in real programs.

## Topics Covered
- String basics and creation
- String indexing and slicing
- String methods (upper, lower, find, replace, strip, split)
- String formatting (f-strings, format())
- String operations (concatenation, repetition)
- Checking string properties (in, startswith, endswith)
- Iterating through strings

## Key Concepts
- **String**: A sequence of characters enclosed in quotes
- **Index**: Position of a character (starts at 0)
- **Slice**: A portion of a string using [start:end]
- **Immutable**: Strings cannot be changed in place
- **Method**: A function that belongs to a string object

## Common String Methods
| Method           | Description                          |
|-----------------|--------------------------------------|
| upper()          | Convert to uppercase                 |
| lower()          | Convert to lowercase                 |
| strip()          | Remove whitespace from both ends     |
| split(sep)       | Split into list by separator         |
| find(sub)        | Find index of substring (-1 if not found) |
| replace(old,new) | Replace occurrences                  |
| startswith(s)    | Check if string starts with s        |
| endswith(s)      | Check if string ends with s          |
