# Chapter 15: Databases and SQL

## Overview
Databases allow programs to store and retrieve large amounts of data persistently. Python's sqlite3 module provides a built-in, file-based relational database.

## Topics Covered
- What is a relational database?
- SQLite and the sqlite3 module
- Creating tables with CREATE TABLE
- Inserting data with INSERT
- Querying data with SELECT
- Filtering with WHERE
- Updating and deleting records
- Database design and normalization
- One-to-many and many-to-many relationships
- JOIN operations

## Key Concepts
- **Database**: An organized collection of structured data
- **Table**: A set of rows and columns (like a spreadsheet)
- **Row/Record**: A single data entry
- **Column/Field**: A specific attribute of data
- **Primary Key**: Unique identifier for each row
- **Foreign Key**: Reference to a primary key in another table
- **SQL**: Structured Query Language
- **CRUD**: Create, Read, Update, Delete operations

## SQL Commands
| Command | Description              |
|---------|--------------------------|
| CREATE TABLE | Create a new table   |
| INSERT INTO  | Add new row          |
| SELECT       | Read/query data      |
| WHERE        | Filter rows          |
| UPDATE       | Modify existing row  |
| DELETE       | Remove rows          |
| JOIN         | Combine tables       |
