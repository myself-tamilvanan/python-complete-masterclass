# Chapter 12: Networked Programs

## Overview
Python can communicate over networks using sockets. This chapter covers the fundamentals of network programming, HTTP, and web scraping.

## Topics Covered
- TCP/IP sockets
- HTTP protocol basics
- Writing a simple web browser with sockets
- Using urllib to fetch web pages
- Parsing HTML with BeautifulSoup
- Unicode and character encoding (UTF-8)
- Application protocols

## Key Concepts
- **Socket**: An endpoint for network communication
- **HTTP**: HyperText Transfer Protocol (how web works)
- **Request/Response**: Client sends request, server sends response
- **urllib**: Python built-in library for URL handling
- **BeautifulSoup**: HTML parsing library
- **UTF-8**: Universal character encoding
