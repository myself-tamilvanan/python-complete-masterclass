## Overview
Matplotlib is the most widely used Python data visualization library. It creates publication-quality charts, plots, and figures with full customization control.

## Installation
```bash
pip install matplotlib
```

## Chart Types
| Chart       | Best For                                |
|-------------|----------------------------------------|
| Line plot   | Trends over time                       |
| Bar chart   | Category comparisons                   |
| Histogram   | Distribution of values                 |
| Scatter plot| Relationship between two variables     |
| Pie chart   | Parts of a whole                       |
| Box plot    | Distribution and outliers              |
| Heatmap     | 2D data intensity                      |
| Subplots    | Multiple charts in one figure          |

## Key Concepts
- **Figure**: The entire image
- **Axes**: A single plot within a figure
- **plt.show()**: Display the plot
- **plt.savefig()**: Save to file
- **plt.style.use()**: Apply built-in styles
